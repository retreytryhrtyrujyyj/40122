https://bysj.bsitem.com/#/home/goodDetail?goodId=40122

## 技术介绍
python语言+Django框架(后端)+Mysql(数据库)+Vue前端框架

## 开发工具
Pycharm+Navicat(mysql数据库可视化工具)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
